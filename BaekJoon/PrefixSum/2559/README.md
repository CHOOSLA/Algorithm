# [BOJ] 2559: 수열

## 문제 정보
- **날짜**: 2026-04-09
- **링크**: [https://www.acmicpc.net/problem/2559](https://www.acmicpc.net/problem/2559)
- **분류**: 구간 합 (Prefix Sum) / 슬라이딩 윈도우 (Sliding Window)

## 문제 요약
- 매일 측정한 온도가 주어졌을 때, 연속적인 K일 동안의 온도의 합이 가장 큰 값을 구하는 문제.

## 힌트
1. 매번 K개의 합을 새로 구하면 $O(N \times K)$의 시간이 걸려 비효율적임.
2. 슬라이딩 윈도우 기법을 사용하면 $O(N)$에 해결 가능함.
3. 처음에 0번부터 K-1번까지의 합을 구한 뒤, 윈도우를 한 칸씩 밀면서 (새로운 값 추가 - 이전 값 제거)를 반복하여 합을 갱신함.
4. 최댓값을 `Integer.MIN_VALUE` 또는 가능한 최소 합으로 초기화해야 함.
