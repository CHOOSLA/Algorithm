# [BOJ] 11657: 타임머신

## 문제 정보
- **날짜**: 2026-04-09
- **링크**: [https://www.acmicpc.net/problem/11657](https://www.acmicpc.net/problem/11657)
- **분류**: 벨만-포드 (Bellman-Ford)

## 문제 요약
- N개의 도시 사이에 M개의 노선이 있을 때, 1번 도시에서 출발하여 다른 모든 도시로 가는 최단 경로를 구하는 문제.
- 음수 가중치가 존재하며, 음수 사이클이 있는 경우 -1을 출력해야 함.

## 힌트
1. 벨만-포드는 음수 가중치가 있는 그래프에서 최단 거리를 구할 때 사용함.
2. 모든 간선을 (N-1)번 반복해서 확인하며 거리를 갱신함.
3. N번째 반복에서도 거리 갱신이 일어난다면 음수 사이클이 존재하는 것임.
4. 출력 시 `long` 타입을 사용하여 오버플로우를 방지해야 함.
