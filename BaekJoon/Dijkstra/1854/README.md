# [BOJ] 1854: K번째 최단경로 찾기

## 문제 정보
- **날짜**: 2026-04-09
- **링크**: [https://www.acmicpc.net/problem/1854](https://www.acmicpc.net/problem/1854)
- **분류**: 다익스트라 (Dijkstra)

## 문제 요약
- 1번 도시에서 각 도시로 가는 최단 경로 중 'K번째'로 짧은 경로의 길이를 구하는 문제.

## 힌트
1. 기본 다익스트라는 각 노드 당 최단 거리 하나만 유지하지만, 여기서는 각 노드 당 K개의 최단 거리를 유지해야 함.
2. 각 노드에 대해 최대 힙(PriorityQueue)을 사용하여 상위 K개의 거리를 관리함.
3. 현재 노드에 대해 새로운 경로의 거리가 힙의 루트(K번째 최단 거리)보다 작으면, 루트를 제거하고 새로운 거리를 추가함.
4. 도착 시 힙의 크기가 K라면 루트 값을 출력, K보다 작으면 -1을 출력함.
